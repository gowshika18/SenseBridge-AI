# SenseBridge AI 🚀

An AI-powered assistive system designed to support **visually impaired 👁️ and hearing impaired 👂 individuals** through real-time communication and environmental awareness.

---

## 🌟 Features

### 👁️ For Visually Impaired
- 🔊 Text-to-Speech (TTS)
- 👁️ Object Detection using Camera
- 📷 Real-time environment capture

### 👂 For Hearing Impaired
- 🎤 Speech-to-Text (STT)
- 📝 Live transcription of voice input

---

## 🧠 Technologies Used

- Python 🐍
- Flask 🌐
- OpenCV 👁️
- SpeechRecognition 🎤
- pyttsx3 🔊
